# レシピくん概要
料理のレシピのURLを投げると、内容をフォーマットしてdiscordに投げてくれる静的サイトです。

自炊を始めたけど、レシピを調べるだけで疲れてしまう。
レシピのサイトが広告だらけだったり、デザインがイカれてたり、３人前だったり４人前だったり。
料理する前に疲れてしまうので、レシピを投げたら1人用の決まったフォーマットにしてくれるページを作りたかった

# 使用方法

(初回起動時のみ)
- webページにアクセス(https://chikuwakun.github.io/shitaaji_resipi_kun/)
- gemini APIキーとwebhook APIキーを設定

- discordに投げたいURLを入れる
- 解析していい感じになる

# レシピのフォーマットについて
以下のフォーマットで出力されるようになっています。
```
【名前】
（レシピの料理名）

【必要なもの（一人前）】

＜食材＞
・食材名 分量
・食材名 分量
（以下必要な分だけ）

＜調味料＞
・調味料名 分量
・調味料名 分量
（以下必要な分だけ）

【調理過程】
1. （手順を簡潔に）
2. （手順を簡潔に）
（以下必要な分だけ）

参考URL: ${url}`;
```

具体的なプロンプトは実装読んでみてください
